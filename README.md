# Library Management System v0

This is a basic modular Python project for managing books, users, and loans in a small library system.

The goal of this project is to practice functions, modules, data structures, and basic project organization before moving into SQL and backend development.

---

## Main Features

* Register books
* List books
* Search books by title
* Register users
* List users
* Borrow books
* Return books
* Track active loans
* Use modular Python files

## Purpose

This project is part of my backend learning path.

The focus is not on databases yet. The first version stores data in memory using lists and dictionaries.

This allows me to practice business logic before adding persistence with SQL in a future version.
